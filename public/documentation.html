My Movie API

This API will provide users with access to information about different movies and the directors and genres of those specific movies. the user will also be able to sign up, update personal information, and create lists of their favorite movies within the app.

Logic URL Method Type Query Parameters Request Body Data Format Response Body Data Format
Returns a list of all movies to the user /movies GET None None JSON object holding data of all movies
Returns data about a single movie by title to the user /movies/:title GET :title None JSON object holding data about a specific movie (e.g. description, genre, director, IMG, and featured)
Returns data about a genre by name/title (e.g. "thriller") /movies/genres/:genre GET :genre None JSON object holding data about a specific genre
Returns data about a specific director /movies/directors/:directorName GET :directorName None JSON Object holding data about a specific director (e.g. Bio, birth year, death year)

Allows users to add a movie to their favorites list /users/:username/favorites/:title POST :username, :title JSON object with move title to add to favorites list JSON object showing movie title added in favorites list
Allow users to delete a movie from their favorites list /users/:username/favorites/:title DELETE :username, :title JSON object with movie title to delete from their favorites list JSON object with the movie title removed from their favorites list
Allow existing users to deregister /users/:username DELETE :username None Confirmation text that account was deleted (200 if successful or 400 if failed
